In order to run this project, there are a few things needed.

Hardware

  • Arduino Nano
  • Ultrasonic Sensor
  • Piezo Buzzer
  • Screen
    • LCD Screen or
    • Seven Segment Display
  • Jumper Wires
  • Variable Resistor
  • 8 220 Ohm Resistors
  • Mini-USB

Software

  • Arduino IDE

Hooking up the components can be done according to the Pin Tables on LCD Project and Seven Segment Display Project.
